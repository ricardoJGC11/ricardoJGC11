Sistema de Gestión de Mantenimiento Computarizado desarrollado con **FastAPI**, **React + Vite** y **PostgreSQL**. Permite administrar activos, órdenes de trabajo y el historial de mantenimiento de una planta mediante una arquitectura cliente-servidor.

### BotE-Commerce

Sistema de comercio electrónico desarrollado con **Python**, **FastAPI**, **PostgreSQL** y un **Bot de Telegram**. Permite gestionar productos y pedidos a través de una API REST integrada con Telegram.
